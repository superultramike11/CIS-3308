Question 3: Clear and Redisplay Data

Requirements

  • Function MakeFilter already takes in an array of character strings and returns a container div. MakeFilter already puts an inner div inside the container div. Modify function MakeFilter so that the inner div has one paragraph per element of the input array. Set the innerHTML of each paragraph to be the corresponding String element from the array.
    • MakeFilter has some sample code that reminds you how to create a paragraph, create a button, set innerHTML, add onclick event, and append to a parent element. Feel free to use or replace this code.
  • After the inner div, the container div shall hold two buttons: "Clear" and "Redisplay".
  • When the user clicks the "Clear" button, the inner div shall be emptied out (innerDiv.innerHTML = "").
  • When the user clicks the "Redisplay" button, the inner div shall again be repopulated (as it was initially). Do not copy paste. Use a single private function (to populate the innerDiv with paragraphs) and call it twice (once initially and again from the Redisplay button).
  • There's no need to modify anything from this index page. All of your work will be done in MakeFilter.js.
