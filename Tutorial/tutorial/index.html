MakePost Component

For this tutorial I created a Post component, to represent news articles posted on my website. This component can be created by calling the MakePost function and passing it two arguments, a post object, and a style object. You can see how these objects should look below:

postObj:
  • articleTitle: The title of the article you'd like to create.
  • articleBody: The articles actual content.
  • articleThumbnail: A link to an image associated with your article.
  • articleImpact: A number ideally between 1-10 that describes how important you as the author, think the story is.
  • articlePostDate: A date ("April 3, 2021, 19:00:00") that represents when an article was first posted. This value would be provided and maintained by the database.
  • articleUpdateDate: A date ("April 3, 2021, 19:00:00") that represents when a future update to the story will be released.
  • articleAuthor: The author of the article.
  • articleAuthorDescription: A little bit of information about the author.
  • articleUpdateBody: The new article body that should appear when a scheduled update time is reached.
styleObj:
  • style: The name of the style class you would like to apply. (Note: This style only effects the article thumbnail.)
Public Functions:
  • setImpactSeverity(severity): Accepts a number, ideally between 1-10, and upates the impact severity and style of the post.
  • updatePost(date, body): Accepts a date in the future (In the format of: "April 3, 2021, 19:00:00") as well as a body of text, and updates the post body when the passed date is reached.

Code Snippets

  • Alert Buttons
    •    This Alert Buttons snippet is what I used to display the impact of a story on my widget. These colorful buttons are a great way to grab someones attention. I was able to achieve this through a super simple process. While I'm updating the impact level, I check if the current level falls withing the range of a different rating. If it does, I update the innerHTML of the button as well as the style, which indicates the stories rating.
  • Countdown
    •    This Countdown snippet was the most interesting for me to work on. This snippet displays a dynamic countdown to a future date that the user enters. It's pretty simple within this code snipit, but gets a little more difficult in my actual tutorial where the date can be updated in different ways. All we do to achieve this is create a function that runs the setInterval callback every 1 second, and then within setInterval callback, we calculate the difference between the current time and the time the user entered. Once we have that difference, we just pass it into a string building function that makes our output look nice. The innerHTML gets updated with every setInterval call, which is what gives this snippet its real-time effect.
  • Read More
    •    I added this snippet of code to my project last minute, in order to keep my components looking nice. This snippet breaks down a large chunk of text into visible and hidden sections, then places a clickable button that allows you to toggle the hidden sections visibility. We can achieve this by slicing the text into our desired sizes, then placing each half into a span, and creating another span for the clickable Read More / Read Less text. The span with the desired hidden half gets the style display:none, by default. When the Read More or Read Less span is clicked, we toggle the hidden span's style to display:inline.
