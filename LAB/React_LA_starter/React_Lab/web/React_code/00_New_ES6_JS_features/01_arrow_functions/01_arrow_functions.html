In addition to just "being shorter" notation, there is an important difference between arrow functions and "regular functions".

  • In regular functions, the keyword "this" represents the object that called the function, e.g., a button that was clicked.
  • With arrow functions, the keyword "this" always represents the object that defined the arrow function.
