This version uses the sort method that is built into any JS array. The call to invoke the JS sort method looks like this: arrayName.sort(func) where func is a function that takes two input parameters (both inputs are an element from the array) and returns one of three possibilities: 1 (if first object > second), -1 (if first object < second), 0 (otherwise, objects equal).

This sort is also just an alphabetic sort (no type conversion). I have added a "sort by price" button to show the problem. We'll add type conversion to a future version of sort.
